# 📦 File Packer & Unpacker (Python GUI)

A simple and beginner-friendly **File Packer & Unpacker** application built using **Tkinter (Python GUI library)**.
This tool allows users to **compress (pack)** a folder into a `.pak` file and **extract (unpack)** it back easily.

---

## 🚀 Features

* 📁 Select any folder and pack it into a `.pak` file
* 📂 Unpack `.pak` files back into original files
* 🖥️ User-friendly GUI using Tkinter
* 📊 Progress bar for packing/unpacking
* ✅ Error handling with pop-up messages

---

## 🛠️ Tech Stack

* Python 🐍
* Tkinter (GUI)
* OS module (file handling)

## ⚙️ How It Works

### 🔹 Packing Process

1. Select a folder
2. Choose output `.pak` file
3. Click **Start Packing**
4. Files are compressed into one file

### 🔹 Unpacking Process

1. Select `.pak` file
2. Click **Start Unpacking**
3. Files are extracted
